Automation Flow:
  1. Frontend validates inputs
  2. Frontend sends POST request to backend
  3. Backend validates file (must be PDF)
  4. Backend launches Playwright
  5. Browser navigates to job URL
  6. Browser scans for form elements
  7. Browser detects and fills fields
  8. Browser uploads resume
  9. Browser clicks submit button
  10. Browser checks for success
  11. Browser closes
  12. Backend returns JSON response
  13. Frontend displays results
  14. Backend cleans up temp files
